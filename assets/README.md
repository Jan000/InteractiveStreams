# Assets Directory

This directory holds game assets:

- `fonts/` – TrueType fonts for text rendering (e.g., names, HUD elements)

Place `.ttf` font files in `fonts/` and configure the path in `config/default.json`.

A good free font to use: [Inter](https://fonts.google.com/specimen/Inter) or [JetBrains Mono](https://www.jetbrains.com/lp/mono/).
